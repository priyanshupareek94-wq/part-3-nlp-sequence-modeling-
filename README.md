# Part 3: NLP and Sequence Modeling Mini Project

## Project Overview
This project involves building an NLP pipeline to classify customer support messages into Positive, Negative, or Neutral sentiments.

## Tasks Performed
- Task 1: Data Understanding: Analyzed 1500 records and class distribution.
- Task 2: Preprocessing: Lowercasing, noise removal, and tokenization.
- Task 3: Vectorization: Used TF-IDF to convert text into numerical format.
- Task 4: Baseline Model: Built a Logistic Regression model with 100% accuracy.
- Task 5: Sequence Modeling: Designed an LSTM architecture for sequence learning.
- Task 6: Reflection: Documented the importance of LSTMs and Transformers.

## Repository Structure
- `notebook.ipynb`: Full code and analysis.
- `results/`: Contains model evaluation metrics and charts.
- `requirements.txt`: List of dependencies.
